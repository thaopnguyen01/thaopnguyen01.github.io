# images/

Drop site images here. Suggested files:

- `avatar.jpg` — square portrait for the About page (replace the gradient circle)
- `travel/kyoto.jpg`, `travel/lisbon.jpg`, etc. — replace the gradient tiles on `travel.html`
- `og-image.jpg` — 1200×630 social-share image (optional)

For travel tiles, you can swap the `<figure class="tile">` background by adding
`style="background-image: url('images/travel/kyoto.jpg'); background-size: cover; background-position: center;"`
to the tile and removing its gradient class override in `css/style.css` if needed.
